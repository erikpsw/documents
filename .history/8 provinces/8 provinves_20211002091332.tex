\\(2)\textcolor{red}{证法1:(同除$e^x$变形)}：${e^x} + \sin x + \cos x \ge ax + 2 \Leftrightarrow h(x) = \frac{{ax + 2 - \sin x - \cos x}}{{{e^x}}} \le 1$
